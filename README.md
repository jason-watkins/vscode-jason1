# Jason1

A dark theme for Visual Studio Code derived from Dark Modern, with a warm near-black background and a curated palette emphasizing readability and color harmony.

## Color Palette

| Role | Color |
|---|---|
| Background | `#0E0E0D` |
| Foreground | `#fbfbe7` |
| Keywords | `#EDB32D` amber |
| Strings | `#C0C340` yellow-green |
| Numbers | `#CE3C2B` red |
| Types | `#A1C3C8` muted teal |
| Functions / Variables | `#fbfbe7` warm cream |
| Comments | `#83B172` sage green |
| Preprocessor | `#AAA1B9` muted lavender |
| Regex | `#AAA1B9` muted lavender |
| Selection | `#458588` teal |
| Accent | `#4A8E92` teal |

## Installation

Install from the VS Code Marketplace, or install the `.vsix` directly:

```
code --install-extension jason1-<version>.vsix
```

Then select the theme via **Preferences: Color Theme** (`Ctrl+K Ctrl+T`) and choose **Jason1**.
